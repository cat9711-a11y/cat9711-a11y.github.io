# GitHub Pages Portfolio (MSc Accounting and Analytics)

This folder contains a static portfolio website ready for GitHub Pages.

## Files

- `index.html`
- `styles.css`
- `script.js`
- `.nojekyll`

## Quick Customize

Update these parts in `index.html`:

- Name: `Minji Kim`
- Email: `your.email@example.com`
- GitHub/LinkedIn links
- Project titles and descriptions
- Timeline entries (school, internship, dates)

## Publish to `username.github.io`

1. Create a new repository named `username.github.io` (replace `username` with your GitHub ID).
2. Upload all files from this folder to the repository root.
3. Commit and push.
4. Open:
   `https://github.com/<username>/<username>.github.io/settings/pages`
5. Under **Build and deployment**:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/(root)`
6. Wait a few minutes, then visit `https://username.github.io`.

## If You Want to Keep This in a Normal Repo

Use one of these:

- Publish from repository root (`/(root)`), or
- Publish from `/docs` folder (move these files into `docs/`).
